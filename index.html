Price Match Calculator (US Market)

This calculator was developed for Tier 2 agents as many of them were struggling with price match calculations. Javascript and CSS embedded into the file that could be sent via internal email. Agents are unable to save anything on a permanent basis (bookmarks, files, etc..). Every state in the USA has a different tax rate.

Supervisor Helper - Excel Manipulation

Extension of MS Excel ribbon control with custom actions that are used on a daily basis to manage the floor. Actions include manipulation of Excel worksheet data, MS Outlook calls, application launch and triggering specific actions in external applications.

Call Helper - Excel Dynamic Content

Developed while working for InfoSys as they have a completely paperless environment and bookmarks in address bar are locked out via GPO. Excel document that has 3 major settings categories
- Credential Manager - place to store all of your login details
- Navigation Manager - place to create drop down menus and assign desired credentials
- Content Manager - place to save useful information / reminders / unique problem - solution cases

Once information is populated, pressing the button Create HTML will create an HTML page with content specified in Excel. This page can then be kept in the mailbox for easier access.
Includes also a feature to phonetically spell out entered text, and all passwords / usernames and content from the dropdown menus is copied to clipboard on click.
